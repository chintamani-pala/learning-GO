# Practice Questions: Functions

**Topic:** Function Declaration & Parameters

1. Write a function named `greet` that takes a `name` string as a parameter and prints a greeting.
2. Create a function that takes two integers and returns their product.
3. Can a Go function return multiple values? Write a function that returns both the area and perimeter of a rectangle.
4. What is the difference between a function parameter and a function argument?
5. Write a function that takes a slice of integers and returns the average (as a float).
