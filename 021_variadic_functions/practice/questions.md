# Practice Questions: Variadic Functions

**Topic:** Variable Number of Arguments

1. Write a function `sumAll` that takes a variadic number of integers and returns their sum.
2. How do you pass a slice of integers as an argument to a variadic function?
3. Can a function have more than one variadic parameter? Explain why or why not.
4. Write a function that takes a prefix string and a variadic number of strings, then prints the prefix followed by all the strings joined together.
5. Where must the variadic parameter `...T` be placed in the function signature if there are other parameters?
