# Practice Questions: Function Named Return Values

**Topic:** Clean Returns with Named Variables

1. Rewrite a function that returns two values to use "named return values".
2. What is a "naked return" and when should you use it (or avoid it)?
3. Does naming the return values affect how the function is called?
4. Write a function `split(sum int) (x, y int)` that returns two numbers that add up to the sum, using named return values.
5. Explain how named return values can improve the documentation of a function.
